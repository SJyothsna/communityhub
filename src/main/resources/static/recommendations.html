📋 Controller Overview

Base Path: /api/recommendations

This controller manages user recommendations for services. Users can recommend services they've used and found helpful.

GET /api/recommendations
Get all recommendations
Retrieves a list of all service recommendations in the system.
GET /api/services/{serviceId}/recommendations
Get all recommendations for a service
Retrieves all recommendations for a specific service.
GET /api/recommendations/{id}
Get recommendation by ID
Retrieves a specific recommendation by its ID.
POST /api/recommendations
Create a new recommendation
Creates a new recommendation for a service.
GET /api/services/{serviceId}/recommendations/count
Get recommendation count for a service
Returns the total number of recommendations for a specific service.
DELETE /api/recommendations/{id}
Delete a recommendation
Permanently deletes a recommendation from the system.

⚠️ Note

This controller needs to be implemented in the backend. The endpoints shown here are the standard operations that should be created for managing recommendations.
