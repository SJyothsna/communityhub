💡 Getting Started

  1. Start the Spring Boot application: mvn spring-boot:run
  2. Access the H2 console to insert test data or verify the database schema
  3. Use the Community Services test page to test the implemented endpoints
  4. Implement additional controllers as needed for complete functionality
