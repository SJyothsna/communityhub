📋 Controller Overview

Base Path: /api/categories

This controller manages top-level service categories like "Home Services", "Kids Classes", "Health & Wellness", etc.

GET /api/categories
Get all categories
Retrieves a list of all service categories in the system.
GET /api/categories/{id}
Get category by ID
Retrieves a specific category by its ID.
POST /api/categories
Create a new category
Creates a new service category with the provided information.

Category Types: SERVICE, CLASS, HEALTH, EDUCATION, ENTERTAINMENT

PUT /api/categories/{id}
Update an existing category
Updates category information for the specified ID.
DELETE /api/categories/{id}
Delete a category
Permanently deletes a category from the system.

⚠️ Note

This controller needs to be implemented in the backend. The endpoints shown here are the standard CRUD operations that should be created.
