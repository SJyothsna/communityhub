📋 Controller Overview

Base Path: /api/communities

This controller manages community records. Communities are groups of users who share services.

GET /api/communities
Get all communities
Retrieves a list of all communities in the system.
GET /api/communities/{id}
Get community by ID
Retrieves a specific community by its ID.
POST /api/communities
Create a new community
Creates a new community with the provided information.
PUT /api/communities/{id}
Update an existing community
Updates community information for the specified ID.
DELETE /api/communities/{id}
Delete a community
Permanently deletes a community from the system.

⚠️ Note

This controller needs to be implemented in the backend. The endpoints shown here are the standard CRUD operations that should be created.
